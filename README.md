A simple web application built with Django that allows users to view meetups, and register for events.  
All meetups are managed through the Django admin panel, including image uploads and organizer information.

User Features
- Open detailed meetup pages.
- Contact organizer via email link.
- Register for a meetup using an email form.
- Confirmation page after successful registration.
- Images displayed for each meetup (uploaded by admin).

Admin Features
- Create and manage meetups in Django admin.
- Upload meetup images.
- Add organizer contact email.
- View list of registered participants.
